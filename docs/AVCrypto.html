AVCrypto

public class AVCrypto
This class contains utility methods for interacting with the Alpha Vantage cryptocurrency API. The methods and parameters available on the Alpha Vantage API are listed in the Alpha Vantage documentation: www.alphavantage.co/documentation/.

Methods

  • daily
    Get daily time series data for a digital/crypto currency.
  • exchangeRate
    Get exchange rate between two currencies.
  • monthly
    Get the monthly time series data for a digital/crypto currency.
  • weekly
    Get the weekly time series data for a digital/crypto currency.
